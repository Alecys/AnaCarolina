# Contador de Vendas – Imobiliária

Site estático simples para acompanhar vendas.

## Como usar no GitHub Pages
1. Crie um repositório.
2. Envie `index.html`, `style.css` e `script.js`.
3. Vá em **Settings > Pages** e ative.
4. Acesse o link gerado.

## Controles
- **Vendeu +1**: soma uma venda.
- **Desvendeu -1**: botão oculto (aparece ao passar o mouse).
- **Atalho secreto**: segure **ALT** e clique no número para diminuir.

O valor fica salvo no navegador (localStorage).
